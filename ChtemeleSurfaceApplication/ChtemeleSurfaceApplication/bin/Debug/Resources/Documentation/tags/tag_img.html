La balise <img>

<img> insere une image sur la page.
<img> à deux attributs requis: src et alt.
Note: les images ne sont pas vraiment insérés dans la page HTML, mais linké aux pages HTML. Une zone se créee pour l'image référencée.

Attribute (spécifique à la balise) Value Description
alt text Spécifie un texte pour l'image
src URL Spécifie l'URL de l'image
height pixels Spécifie la hauteur de l'image
width pixels Spécifie la largeur de l'image
Attribute (géneraux) Description
Class Spécifie un ou plusieurs nom(s) de classe pour un element(réfere à une classe dans le css)
id Spécifie un id unique pour un element
style Spécifie un style css sur une ligne
title Spécifie des informations à propos de l'élement
