La balise <em>

La balise <em> désigne une emphase (une mise en relief, une insistance) et affiche le texte qu'elle contient dans la plupart des navigateurs en italique.
Ce tag applique ainsi un effet équivalent à la balise <i> qui affiche aussi son contenu en italique ou encore au style CSS "font-style : italic;".
Elle diffère cependant de part la sémantique qui lui est associée qui a pour effet de donner du poids au texte contenu au niveau des moteurs de recherche (pour le référencement).

Attribute (géneraux) Description
Class Spécifie un ou plusieurs nom(s) de classe pour un element(réfere à une classe dans le css)
id Spécifie un id unique pour un element
style Spécifie un style css sur une ligne
title Spécifie des informations à propos de l'élement
