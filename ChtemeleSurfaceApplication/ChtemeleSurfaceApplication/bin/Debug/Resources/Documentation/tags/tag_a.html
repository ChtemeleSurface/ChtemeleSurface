La balise <a>

<a> définit un hyper-lien, qui est utilisé pour renvoyer sur une page en cliquant dessus.
L'attribut le plus important de la balise <a> est le href, qui indique la destination.
Par défaut, les liens apparaissent comme ceci dans les differents navigateurs:
Un lien non visité est souligné et de couleur bleu.
Un lien visité est souligné mais de couleur violette.
Un lien actif est souligné mais de couleur rouge.

Attribute (spécifique à la balise) Value Description
href URL Spécifie l'URL de la page auquel le lien renvoie (nous emmènes)
download filename Spécifie la cible de l'hyperlien a telecharger
hreflang language_code Spécifie le langage du document
media media_query Spécifie le media pour lequel le document est optimisé
rel alternate
autdor
bookmark
help
license
next
nofollow

tag		 Spécifie les relations entre le document ouvert et le document attaché
target _blank
_parent
_self
_top
framename		 Spécifie ou ouvrir le document
type MIME_type Spécifie le MIME type du document attaché
