La balise <header>

Définition:

<header> definit une entete pour un document ou une section.
<header> doit être utilisé en tant que conteneur pour un contenu introductif en haut de page.
On peut avoir plusieurs <header> dans un seul document.
Note: Un <header> ne peut être placé sans un <footer>, un <adress> ou un autre <header>.

Attributs:

Attribute (géneraux) Description
Class Spécifie un ou plusieurs nom(s) de classe pour un element(réfere à une classe dans le css)
id Spécifie un id unique pour un element
style Spécifie un style css sur une ligne
title Spécifie des informations à propos de l'élement

Exemple :


Ceci est un header
			<header>
				Ceci est un header
			</header>
