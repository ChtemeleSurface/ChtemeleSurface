La balise <blockquote>

Définition:

<blockquote> definit une section qui est tirée d'une autre source.
Les navigateurs indentent automatiquement les elements de type <blockquote>.

Attributs:

Attribute (spécifique à la balise) Value Description
cite URL Spécifie la source de la citation


Attribute (géneraux) Description
Class Spécifie un ou plusieurs nom(s) de classe pour un element(réfere à une classe dans le css)
id Spécifie un id unique pour un element
style Spécifie un style css sur une ligne
title Spécifie des informations à propos de l'élement

Exemple :


Ceci est une citation
			<blockquote>
				Ceci est une citation
			</blockquote>
